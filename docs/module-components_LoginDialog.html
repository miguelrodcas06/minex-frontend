Module: components/LoginDialog

Diálogo modal de autenticación de MineX. Permite al usuario alternar entre las pestañas de "Iniciar Sesión" y "Registrarse". Tras un login exitoso almacena el token en `sessionStorage`, emite el evento `minex:login` y llama al callback `onLoginSuccess` del padre.
