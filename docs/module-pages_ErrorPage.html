Module: pages/ErrorPage

Página de error polivalente para MineX. Detecta el tipo de error (404, 503, red, genérico) y muestra un mensaje contextualizado con icono, código HTTP y botones de "Reintentar" / "Ir al inicio". En entorno de desarrollo (`import.meta.env.DEV`) también muestra el stack del error.
