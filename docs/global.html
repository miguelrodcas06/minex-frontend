generatePDF(zonaImpresion, nombreDocumento) → {void}

Genera y descarga automáticamente un PDF a partir de un elemento del DOM. Captura el elemento con `html2canvas` (escala ×2 para mayor resolución) omitiendo los elementos con la clase `omitir-pdf`, y genera un documento A4 con `jsPDF`.
Parameters:
Name Type Description
zonaImpresion string ID del elemento HTML a capturar.
nombreDocumento string Nombre del archivo PDF generado (sin extensión).
