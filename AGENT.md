# Gestión Cultural Frontend

Frontend del proyecto de gestión cultural.

### Contexto del proyecto

Este frontend usa una BD phpmyadmin dockerizada. La conexion funciona poniendo primero en pie el backend que es
minex-backend usando npm run dev

### Resctricciones

- NO modifiques nada del backend minex-backend
- Solo modifica el proyecto minex-frontend usando react y la estructura
base que ya esta creada, viendo la manera de operar (añadir el componentes en componet usando axios, exportala a App...)
