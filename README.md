## Ejecución del Proyecto

### Backend

1.  Navegar al directorio del backend:
    bash
    cd ../minex-backend
    
2.  Instalar dependencias:
    bash
    npm install
    
3.  Iniciar el servidor de desarrollo:
    bash
    npm run dev
    
### Frontend

1.  Navegar al directorio del frontend:
    bash
    cd ../minex-frontend
    
2.  Instalar dependencias:
    bash
    npm install
    
3.  Iniciar el servidor de desarrollo:
    bash
    npm run dev
